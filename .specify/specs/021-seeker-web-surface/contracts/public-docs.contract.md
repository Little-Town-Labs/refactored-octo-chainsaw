# Public Docs Contract

## `/agents.md`

Must be publicly readable and include:

- Spyglass overview.
- Human seeker entry points.
- Agent-readable entry points.
- A2A card index and individual card locations.
- Statement that A2A is discovery/future interop and not required for v0 customer flow.
- Unsupported actions: seeker dashboard, ticket list, analytics, recommended jobs, browse jobs, raw dossiers, transcripts, hidden run state, direct employer messaging.
- Governance/source-of-truth pointers.

## `/llms.txt`

Must be publicly readable and include:

- Concise site summary.
- Public paths that may be used for discovery.
- Disallowed scraping or model-training reuse statement.
- Privacy and no-dashboard boundary.
- Pointer to `/agents.md` and the A2A card index.

## Prohibited Public Claims

F21 public docs must not claim:

- External A2A protocol handlers are live.
- A seeker-side API is available.
- A dashboard, ticket list, analytics view, recommended-jobs grid, browse-jobs surface, public seeker profile, raw transcript view, or direct employer messaging exists.
